# VMU Automatic Course Registration

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Platform](https://img.shields.io/badge/Platform-Android%20%7C%20Windows%20%7C%20Linux%20%7C%20iOS-blue.svg)](https://github.com/duykhongphai/VMUAutomaticCourseRegistration)

Ứng dụng hỗ trợ đăng ký môn học tự động cho sinh viên Đại học Hàng Hải (VMU).

## Tính năng

- Đăng nhập tài khoản sinh viên VMU
- Xem danh sách môn học có thể đăng ký
- Chọn lớp học và đăng ký tự động
- Theo dõi trạng thái đăng ký
- Hỗ trợ đa nền tảng: Android, Windows, Linux, macOS

## Tải xuống

### Android

**Yêu cầu hệ thống:**
- Android 7.0 (API 24) trở lên
- Dung lượng trống: ~50 MB

**Hướng dẫn cài đặt:**

1. Tải file APK từ [Releases](https://github.com/duykhongphai/RegisterVMUCourses/releases/tag/release)
2. Bật "Cài đặt từ nguồn không xác định" trên thiết bị
   - Vào Cài đặt → Bảo mật
   - Bật "Nguồn không xác định" hoặc "Cài đặt ứng dụng không xác định"
3. Mở file APK đã tải và nhấn Cài đặt
4. Chờ quá trình cài đặt hoàn tất

### Desktop

**Yêu cầu hệ thống:**

| Hệ điều hành | Yêu cầu |
|-------------|---------|
| Windows | Windows 10 (1809+) hoặc mới hơn |
| Linux | Ubuntu 20.04+, Fedora 36+, hoặc các distro tương đương |

**Windows:**

1. Tải file `.rar` từ [Releases](https://github.com/duykhongphai/RegisterVMUCourses/releases/tag/release)
2. Giải nén file rar
3. Chạy file .exe
4. Làm theo hướng dẫn trên màn hình

**Linux (Ubuntu 24.04):**

1. Tải file `DangKyVMU_App` từ [Releases](https://github.com/duykhongphai/RegisterVMUCourses/releases/tag/release)
2. Cấp quyền thực thi và chạy:
   ```bash
   chmod +x ./DangKyVMU_App
   ./DangKyVMU_App
   ```

### iOS

1. Tải file `.ipa` từ [Releases](https://github.com/duykhongphai/RegisterVMUCourses/releases/tag/release)
2. Sử dụng [SideLoader](https://sideloadly.io/) để cài đặt

## Hướng dẫn sử dụng

**Bước 1: Đăng nhập**
- Mở ứng dụng
- Nhập mã sinh viên và mật khẩu VMU
- Nhấn "Đăng nhập"

**Bước 2: Chọn môn học**
- Xem danh sách môn học có thể đăng ký
- Chọn lớp học mong muốn
- Thêm vào danh sách đăng ký

**Bước 3: Đăng ký tự động**
- Xác nhận danh sách môn học
- Nhấn "Đăng ký" để bắt đầu
- Theo dõi trạng thái đăng ký

## Lưu ý quan trọng

- **Bảo mật:** Ứng dụng không lưu trữ thông tin đăng nhập của bạn trên server
- **Mục đích:** Ứng dụng chỉ dùng để hỗ trợ, không đảm bảo 100% đăng ký thành công
- **Trách nhiệm:** Người dùng tự chịu trách nhiệm khi sử dụng ứng dụng
- **Kết nối:** Cần kết nối internet ổn định khi sử dụng
